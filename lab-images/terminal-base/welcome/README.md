# Welcome to Cosmos Corp

Hey - Sage here. Glad to have you on the team.

This folder is your sandbox for week one. Every file you see here was set up
specifically so you can poke around without breaking anything important.

A few starting points:

- `schedule.txt` - your loose plan for the week.
- `manifest.log` - a long-running build log. Good for practising `head` / `tail`.
- `crew/` - short bios for the people you'll be paged by most.
- `notes/` - empty for now. Yours to fill.

When in doubt, `ls` first, then read. Take your time.
