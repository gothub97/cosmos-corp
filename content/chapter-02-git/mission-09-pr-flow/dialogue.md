# The PR loop, in miniature

**Sage** props themselves up on the desk.

> "Final mission of the chapter. Today you put the whole loop together -
> the way real teams work day to day:
>
> 1. **Branch** off main.
> 2. **Commit** your change.
> 3. **Push** the branch up (with `-u` the first time, so git knows where
>    it lives).
> 4. **Fetch** to see if anyone left feedback.
> 5. **Address** it with another commit.
> 6. **Push** again.
>
> Cosmos has a fake review bot that posts notes on every new branch - it'll
> already have something waiting for you when you push.
>
> No surprises. You've done every individual step before. Just chain them."

---

You start in `~/work/cosmos`, on `main`. Your task is to publish a
`feature/handbook-update` branch with a brand-new `docs/handbook.md` -
then satisfy whatever the bot asks for.
