# Moving things around

**Sage** flips open a fresh Slack thread.

> "OK - reading is great, but you'll spend most of your week *changing*
> files. Here are the four bread-and-butter commands:
>
> - `mkdir` - make a directory
> - `touch` - create an empty file (or update its timestamp)
> - `cp` - copy
> - `mv` - move *or* rename (same command, depends on the destination)
> - `rm` - delete
>
> I left a `~/scratch/` folder for you with a template and an old file.
> See the objectives panel - you'll set up a daily-log structure inside it."

---

> ⚠️ **About `rm`** - there is no recycle bin on Linux. `rm` deletes for real.
> No need to be scared - `~/scratch/` is a sandbox - but it's a habit worth
> building now.
