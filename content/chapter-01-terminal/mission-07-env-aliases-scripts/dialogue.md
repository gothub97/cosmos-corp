# Make the shell yours

**Sage** sits down across from you.

> "Last terminal mission. This one's about *making the shell feel like
> yours*. Three concepts:
>
> - **`export FOO=bar`** - set a variable that any program you run can read.
> - **`alias gs='git status'`** - give a long command a short name.
> - **A shell script** - a file full of commands you can run by name. The
>   tiniest possible one is just a shebang and an `echo`. We'll write one
>   here so you've done it once. After that, every Bash script you'll
>   ever read is just a longer version of the same idea."

---

You'll need an editor for the script - `nano ~/work/hello.sh` is the
friendly choice (Ctrl+O to save, Ctrl+X to quit). `vim` works too if you
already know it.
